# Attribution

This repository began as a fork of an upstream civic-legislator data project. The original direction included legislator organization, jurisdiction/chamber/district/term modeling, PostgreSQL usage, and OpenStates-style parsing.

FLLC / PersonFu later expanded the fork into Arizona ALPR accountability, public safety camera oversight, AZ511 and public-road data continuity, public-records workflows, speed/photo-enforcement court-process documentation, and camera-governance research.

This file keeps upstream project credit visible without adding unnecessary personal-identifying detail.
