# Evidence Standards

Overture separates public statements into four working categories: supported, needs public records, needs location verification, and unsupported or overstated.

A supported statement has a source that directly backs the wording. A statement that needs public records should be written as a question or research task. A location lead should not be treated as final proof. Unsupported wording should not appear as a fact in public-facing material.

Before publishing, check the exact sentence, the source that supports it, the limits of that source, and the next action a resident can take.

Run `python main.py verify` to check source IDs in the current pipeline.
